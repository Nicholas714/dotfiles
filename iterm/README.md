To setup macOS with iTerm:

1) Download iTerm
  - Use chsh to set it as the new shell
2) Download fish https://fishshell.com
3) Download oh-my-fish https://github.com/oh-my-fish/oh-my-fish
 - Use omf install agnoster and omf theme agnoster
4) Download colors from https://github.com/lysyi3m/macos-terminal-themes if you want
 - Just used the MonkaiEdited.json file and import it
5) Setup fonts by downloading https://github.com/powerline/fonts
 - Profile should already use Meslo LG L for Powerline
6) Copy over ~/.config/fish/config.fish
7) iTerm2 > Install Shell Integration 
8) Add Toolbelt, etc.
